# Shell Script
Shell 脚本是用来自动化执行命令的文本文件，通常以 `.sh` 结尾。可用于 Linux、macOS 等类 Unix 系统。
---
## Bash
Bash（Bourne Again Shell）是最常用的 Shell，提供了丰富的脚本功能和命令行交互能力。
**示例：**
```bash
#!/bin/bash
echo "Hello, World!"
```
---
## Shell Variables
shell 变量用于存储值，可以是字符串或数字。
**定义变量：**
```bash
NAME="John"
AGE=25
```
